# Copilot Relay Android

Convert OpenAI format requests to GitHub Copilot API.

Currently supported APIs:

- `POST /v1/chat/completions`

Inspired by [copilot-gpt4-service](https://github.com/aaamoon/copilot-gpt4-service)
